Q11What is useReducer used for?▼
Managing complex state logic with actions and reducers — preferred over useState for complex cases.
Q12What is the purpose of useEffect?▼
To perform side effects (e.g., API calls, subscriptions) in functional components.
Q13What is useContext?▼
A hook for accessing context values without prop drilling — used for global state.
Q14What is prop drilling?▼
Passing props through multiple layers of components that don't need them, just to reach a deeply nested child.
Q15What is Tailwind CSS?▼
A utility-first CSS framework for rapid UI development — uses pre-defined classes directly in HTML/JSX.
Q16What is React Router?▼
A library for client-side navigation in Single Page Applications (SPAs) — enables routing without full page reloads.
Q17What is <Link> used for?▼
